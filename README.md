# swapDate

Simple console app that swaps `yyyy/mm/dd` to `dd/mm/yyyy` in a text file (Whatsapp Export Chat TXT)


Sample Before:
```
2022/07/15, 12:48 - 
2022/07/15, 12:48 - 
2022/07/15, 13:39 - 
2022/07/15, 13:40 - 
2022/07/15, 13:40 - 
2022/07/15, 13:41 - 
```

Sample After:
```
15/07/2022, 12:48 - 
15/07/2022, 12:48 - 
15/07/2022, 13:39 - 
15/07/2022, 13:40 - 
15/07/2022, 13:40 - 
15/07/2022, 13:41 -
```
